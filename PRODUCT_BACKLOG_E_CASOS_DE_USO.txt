ÉPICO 1: AUTENTICAÇÃO E CONTROLE DE ACESSO
===========================================

USER STORIES:
-------------

US001 - Login no Sistema
- Como usuário, quero fazer login no sistema para acessar as funcionalidades
- Critérios de Aceitação:
  * Validação de credenciais (usuário/senha)
  * Redirecionamento baseado no tipo de acesso
  * Mensagens de erro para credenciais inválidas
  * Loading state durante autenticação

US002 - Acesso Diferenciado por Tipo de Usuário
- Como administrador master, quero ter acesso a todas as unidades da usina
- Como operador de filial, quero acessar diretamente minha unidade específica
- Critérios de Aceitação:
  * Usuários master/sede/admin: redirecionam para seleção de unidades
  * Usuários normais: redirecionam diretamente para home
  * Validação automática do tipo de acesso

US003 - Logout Seguro
- Como usuário, quero fazer logout para encerrar minha sessão com segurança
- Critérios de Aceitação:
  * Limpeza da sessão
  * Redirecionamento para página de login
  * Confirmação de logout

FLUXOS PRINCIPAIS:
==================

FLUXO DE AUTENTICAÇÃO:
---------------------
1. Usuário acessa página de login
2. Sistema valida credenciais
3. Sistema identifica tipo de usuário
4. SE usuário master ENTÃO redireciona para seleção de unidade
5. SENÃO redireciona para home da unidade
6. Usuário acessa funcionalidades conforme permissões

FLUXO DE GESTÃO DE ESTOQUE:
--------------------------
1. Operador consulta estoque atual
2. Sistema apresenta níveis de estoque
3. SE estoque baixo ENTÃO sistema gera alerta
4. Operador cria requisição de reposição
5. Supervisor aprova requisição
6. Sistema atualiza status da requisição
7. Operador registra entrada de material
8. Sistema atualiza níveis de estoque

FLUXO DE PRODUÇÃO SMART CALDA:
-----------------------------
1. Operador seleciona receita
2. Sistema verifica disponibilidade de ingredientes
3. Operador confirma carga
4. Sistema registra início da produção
5. Equipamento executa receita automaticamente
6. Sistema monitora progresso
7. Sistema registra conclusão
8. Operador confirma qualidade do produto
